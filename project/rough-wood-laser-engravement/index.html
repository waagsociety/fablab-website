Rough wood laser engravement
Michalis
Thu, 10/22/2015

This project is based on the beauty of contrast. Detailed calligraphy on rough wood.

It's destiny is a name board for a small cottage. An outdoor installation thus. So I thought, the deeper we engrave the better! That way it would take longer for the effect of weather conditions to show their toll and threaten the legibility of this name board.Â 

The wood is hard. And we have set the laser power high, hoping to achieve that desired depth. This has caused some concern as the wood is burning slightly. We have therefore lowered the power. We would need to do it twice or three times in order to get deep. So to do it safely, and get a satisfying result, I think that 4-5 hours will be needed.
