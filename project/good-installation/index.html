Good installation
jord beets
Thu, 08/15/2013

For a project of a non-profit organisation ('Good), I worked on an installation to light up their logo by using muscle power. To create this, I used the Shopbot cnc milling machine.

Although I had some previous experience with these machines, I still had to work a complete morning to get familiar with the software (every machine seems to have a slightly different interface). After some failures, I succeeded in my mission.

For this project I created two identical logos, one in acrylate and one in mdf. I did run into some problem, especially with the acrylate. It proved hard to keep the material flat on the ground surface, it tends to raise a little. The milling machine will then cut deeper at certain places, and gets hot, causing the material to melt. This leaves ugly traces.

In first instance, I had used it in a 'mirrored' fashion so to speak. Small fragments are pushed upwards in this way, but the melted material got stuck this way. Luckily, mdf was easier to work with. With some help, this worked a lot faster.
