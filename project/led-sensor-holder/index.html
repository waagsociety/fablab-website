LED Sensor holder
Lara D
Tue, 01/06/2015

The LED sensor holder is a project started at Open Makers Lab in the Wildemanbuurt.

This holder uses a 3D STL file to drive the CNC machine. It was first thought it would be 3D-printed, we plans were changed and it was fabricated in wood instead.

The measurements were entered in TinkerCAD where the model was digitized and further enhanced.
