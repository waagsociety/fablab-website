For a project for study i needed to mill a 3d model with the Roland Modela (MX-20?).

What I used for material to make the model is in hardness a litle bit softer than soft wood. 

The settings of the milling machine are:
xy speed   15mm/sec (max speed)
Z speed     0.8 mm/sec
Spindle     6500rpm
Cutting amount   1,5mm
Part interval    1,6
finish margin  0,1
Bit 1/8 insch square
 
The speed of the z axes could be a lot quicker. Because it's takes a lot of time and the machine could handle mutch more than that i guess.
Also it would be more eficient to mill instead along the x axle to mill countour lines. 
With the settings now it would take more than 4 hours to mill one side and if you would do the more efficient way it would take 2,5 hours.
