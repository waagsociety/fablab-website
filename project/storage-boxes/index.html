I started a project about storage boxes for kids, because I was not happy with the conventional box standard solutions that are commercially available. This project is trying to push the boundaries of how square a box needs to be. :-)

My design of the storage boxes resembles an 'owl', making it fun to have in a kids room. Also the storage boxes are intended be assembled and dismantled very easy by the parent with minimal storage space. So I'm working on making it a flatpack solution, that can be (dis)assembled without tools.
