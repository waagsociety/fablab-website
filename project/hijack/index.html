I cut and etched 2mm vegetable tanned leather, to prototype a wallet project I've been working on. I used the BRM laser with the following settings:

Cut
speed 100
power 100
corner 30

Engrave
speed 400
power 30

Two rules of thumb that I learned the hard way and are somehow not standard practise when learning the laser:

1) make sure your engrave file runs before the cut file (otherwise the curves in the leather distort your engraving once it is cut free and doesn't lay flat)

2) Save the origin point for the laser so that you can stop and restart a file from the same origin!

It is a really powerful and nice laser, but it can take a long time to cut. The laser doesn't seem to optimize the fasted route for cutting and engraving so small batches seems to work best.
