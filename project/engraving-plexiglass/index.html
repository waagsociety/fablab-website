Engraving plexiglass

Wed, 04/11/2012

I am a graphic designer an I like to create things myself. So you can imagine how happy I was when I heard about Fablab :)

This piece of plexiglass is a little window that separates the two rooms in my house. It measure 77x 33 cm. I wanted to decorate it with jungle motif and I choosed for the laser cutter. It was a little work to understand how it worked (first time), but then the machine started.

It cutted very quickly the outline and than I gave the ok to start the engraving. It took 35 minutes only for the half. Still need to engrave the other half... IÂ used the setup on the reference board in Fablab.

Next time I definitely will take note of it.
